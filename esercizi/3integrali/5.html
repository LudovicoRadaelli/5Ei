Regole di integrazione

  • \[ \int x^{\alpha} \,\,dx = \dfrac{x^{\alpha + 1}}{\alpha + 1} + c \quad \text{con} \quad \alpha \neq -1 \]

  • \[ \int \dfrac{1}{x} \,\, dx = ln\left(x\right) + c \quad \text{con} \quad x \gt 0 \]

  • \[ \int e^{x} \,\, dx = e^{x} + c \]

  • \[ \int sin(x) \,\, dx = -cos(x) + c \]

  • \[ \int cos(x) \,\, dx = sin(x) + c \]

    • Soluzione: \[ -5\,cos(x) -3\,sin(x)+ c \]

  • \[ \int \left(\dfrac{3x - 4x^2 + 5x^3}{2x^2}\right) dx \]

    • Suggerimento:
    Prima di tutto distribuite la frazione sugli addendi del numeratore, poi semplificate ciascuna frazione: \[ \begin{align*} \int \left(\dfrac{3x - 4x^2 + 5x^3}{\color{red}{}2x^2}\right) dx = \\\\ \int \left(\dfrac{3x}{\color{red}{}2x^2} - \dfrac{4x^2}{\color{red}{}2x^2} + \dfrac{5x^3}{\color{red}{}2x^2}\right) dx = \\\\ \int \left(\dfrac{3\cancel{x}}{2x^{\cancel{2}}} - \dfrac{4\cancel{x^2}}{{}2\cancel{x^2}} + \dfrac{5x^{\cancel{3}}}{2\cancel{x^2}}\right) dx = \\\\ \int \left(\dfrac{3}{2x} - \dfrac{4}{2} + \dfrac{5x}{2}\right) dx = \end{align*} \] Ed ora vai di linearità!

    Soluzione: \[ \dfrac{3}{2}\,ln(x) -2x + \dfrac{5}{4}x^2 + c \]
