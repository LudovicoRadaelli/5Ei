Regole di integrazione

  • \[ \int x^{\alpha} \,\,dx = \dfrac{x^{\alpha + 1}}{\alpha + 1} + c \quad \text{con} \quad \alpha \neq -1 \]

  • \[ \int \dfrac{1}{x} \,\, dx = ln\left(x\right) + c \quad \text{con} \quad x \gt 0 \]

  • \[ \int e^{x} \,\, dx = e^{x} + c \]

  • \[ \int sin(x) \,\, dx = -cos(x) + c \]

  • \[ \int cos(x) \,\, dx = sin(x) + c \]

Svolgere i seguenti integrali

  • \[ \int\left(3\,e^{^x} - 3\,cos(x) + 1\right) dx \]

    • Soluzione: \[ 3\,e^{^x} -3 sin(x) + x + c \]

  • \[ \int \left(-\dfrac{2}{x} + \dfrac{3}{x^2}\right) dx \]

    • Soluzione: \[ -2\,ln(x) - \dfrac{3}{x} + c \]

  • \[ \int\left(9 sin(x) - \dfrac{1}{2}cos(x) + 3x^5\right) dx \]

    • Soluzione: \[ -9\,cos(x) - \dfrac{1}{2}sin(x) + \dfrac{x^6}{2} + c \]

  • \[ \int \dfrac{-2x^2 + x^3 -7x^4 }{x^3} \,\, dx \]

    • Soluzione: \[ -\dfrac{7}{2}x^2 + x - 2\,ln(x) \]

  • \[ \int\left(\sqrt[3]{x} - 2\sqrt[5]{x^3} + \dfrac{3}{x^4}\right) dx \]

    • Soluzione: \[ \dfrac{3}{4}\,\sqrt[3]{x^4} - \dfrac{5}{4}\,\sqrt[5]{x^8} - \dfrac{1}{x^3} \]
