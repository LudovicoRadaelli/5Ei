Studiare e memorizzare le seguenti regole di integrazione, che estendono quelle studiate nelle lezioni precedenti.

Regole di integrazione (generalizzate)

  • \[ \int f(x)^{\alpha} f'(x) \,\,dx = \dfrac{f(x)^{\alpha + 1}}{\alpha + 1} + c \quad \text{con} \quad \alpha \neq -1 \]

  • \[ \int \dfrac{f'(x)}{f(x)} \,\, dx = ln\left(|f(x)|\right) + c \]

  • \[ \int e^{f(x)}f'(x) \,\, dx = e^{f(x)} + c \]

  • \[ \int sin(f(x))f'(x) \,\, dx = -cos(f(x)) + c \]

  • \[ \int cos(f(x))f'(x) \,\, dx = sin(f(x)) + c \]

  • \[ \int \dfrac{1}{cos^2(f(x))}f'(x) \,\, dx = tan(f(x)) + c \]

  • \[ \int \dfrac{1}{sin^2(f(x))}f'(x) \,\, dx = -cotan(f(x)) + c \]

  • \[ \int \dfrac{f'(x)}{\sqrt{1 -f(x)^2}} \,\, dx = arcsin(f(x)) + c \]

  • \[ \int \dfrac{f'(x)}{1 + f(x)^2} \,\, dx = arctan(f(x)) \]
